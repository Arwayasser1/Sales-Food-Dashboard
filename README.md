# Sales-Food-Dashboard
Project Title: Food Sales Performance Dashboard

Project Overview: This project focuses on analyzing food sales data to uncover key business trends and performance metrics. By transforming raw transactional data into a dynamic visual dashboard, I provided a clear view of revenue growth, regional dominance, and product demand.

Key Metrics (KPIs):

Total Revenue: $1,034,734

Total Orders: 214

Average Order Value: $4,835

Top Category: Fruits (67.5% of total sales)

Top Market: United States ($267,133)

Dashboard Features & Visuals:

Sales Trends: A line chart analyzing monthly revenue, highlighting a peak in September ($151,549).

Product Performance: A bar chart identifying "Bananas" as the highest revenue-generating product ($345,295).

Regional Insights: A pie chart showing that the "South" region leads with the highest number of orders (93).

Quarterly Analysis: A horizontal bar chart showing Q1 as the strongest period for sales ($416,689).

Dynamic Filtering: Integrated slicers for Year, Category, Product, Country, and Quarter to allow for deep-dive interactive analysis.

Technical Tools Used:

Data Cleaning: Organized and formatted raw CSV/Excel data for accuracy.

Pivot Tables: Used to aggregate and summarize large datasets efficiently.

Data Visualization: Built custom charts and a cohesive UI/UX layout.

Interactive Slicers: Implemented for real-time data exploration and user interactivity.

Business Value: The dashboard enables stakeholders to identify high-performing regions and products quickly, allowing for better inventory management and targeted marketing strategies based on seasonal sales trends.
